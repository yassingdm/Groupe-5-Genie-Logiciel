# Vue Console

Pour compiler le projet:

```bash
cd grp5
mvn clean compile
```

Pour lancer le projet en mode console:

```bash
java -cp target/classes eidd.grp5.app.App
```

Fonctionnalites actuellement disponibles dans la console:
- Afficher les statistiques
- Ajouter un utilisateur
- Lister les utilisateurs
- Ajouter une salle
- Lister les salles
- Creer une reservation simple (utilisateur + salle + dates)
- Lister les reservations
